1. **Clonar o descargar el proyecto**:
   Asegúrate de tener todos los archivos (`app.py`, `models.py`, `templates/`, `static/`) en una sola carpeta.

2. **Preparar el entorno**:
   Se recomienda usar Python 3.12. Instala Flask si aún no lo tienes:
   ```bash
   pip install flask
